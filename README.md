# Full Cycle Rocks - Desafio Nginx + Node.js + MySQL

## Descrição
Este projeto implementa uma aplicação web onde:
1. Nginx atua como proxy reverso na porta 8080
2. Aplicação Node.js cadastra nomes no MySQL
3. A interface exibe "Full Cycle Rocks!" e lista todos os nomes cadastrados

## Como executar

1. Clone o repositório
2. Execute o comando:
```bash
docker-compose up -d
